
# Translate UTF-8 Serbian Latin text to Serbian Cyrillic

Support replace of 2 letters (ASCII charachters) digraphs with one letter UTF-8 characther:

* "lj" -> "ǉ" -> "љ"
* "nj" -> "ǌ" -> "њ"
* "dž" -> "ǆ" -> "џ"
* "Lj" -> "ǈ" -> "Љ"
* "Nj" -> "ǋ" -> "Њ"
* "Dž" -> "ǅ" -> "Џ"
